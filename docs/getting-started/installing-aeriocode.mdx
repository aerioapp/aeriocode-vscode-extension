### Installation Options

-   **VS Code Marketplace (Recommended):** Fastest method for standard VS Code and Cursor users.
-   **Open VSX Registry:** For VS Code-compatible editors like VSCodium.

### 🛠️ VS Code Marketplace: Step-by-Step Setup

Follow these steps to get Aeriocode up and running:

1.  **Open VS Code:** Launch the VS Code application.

    > ⚠️ **Note:** If VS Code shows "Running extensions might...", click "Allow".

2.  **Open Your Aeriocode Folder:** In VS Code, open the Aeriocode folder you created in Documents.
3.  **Navigate to Extensions:** Click on the Extensions icon in the Activity Bar on the side of VS Code (`Ctrl + Shift + X` or `Cmd + Shift + X`).
4.  **Search for 'Aeriocode':** In the Extensions search bar, type `Aeriocode`.
5. **Install the Extension:** Click the "Install" button next to the Aeriocode extension.
6. **Open Aeriocode:**
    - Click the Aeriocode icon in the Activity Bar.
    - Or, use the command palette (`Ctrl/Cmd + Shift + P`) and type "Aeriocode: Open In New Tab" for a better view.
7. **Troubleshooting:** If you don't see the Aeriocode icon, try restarting VS Code.

> ✅ **Pro Tip:** You should see the Aeriocode chat window appear in your VS Code editor!

### 🌐 Open VSX Registry

For VS Code-compatible editors without Marketplace access (like VSCodium and Windsurf):

1. Open your editor.
2. Access the Extensions view.
3. Search for "Aeriocode".
4. Select "Aeriocode" by Aerio and click **Install**.
5. Reload if prompted.

### 👤 Creating Your Aerio Account

Now that you have Aeriocode installed, let's get you set up with your account:

1. **Sign In to Aerio:**
    - Click the **Sign In** button in the Aeriocode extension.
    - You'll be taken to [aerio.bot](https://aerio.bot) to create your account.
2. **Start with Free Credits:**
    - No credit card needed!
3. **Available AI Models:**
    - Aeriocode general coding (recommended for general coding)
    - Aeriocode-DO178C (recommended for aviation related coding)
    - Aeriocode-mini

### 💻 Your First Interaction with Aeriocode

You're ready to start building! Copy and paste this prompt into the Aeriocode chat window:

```
Hey Aeriocode! Could you help me create a new project folder called "hello-world" in my Aeriocode directory and make a simple python code that prints "Hello World"?
```

> ✅ **Pro Tip:** Aeriocode will help you create the project folder and set up your first code!

### 🧩 Tips for Working with Aeriocode

-   **Ask Questions:** If you're unsure about something, ask Aeriocode!
-   **Use Screenshots:** Aeriocode can understand images — show him what you're working on.
-   **Copy and Paste Errors:** Share error messages in the chat for solutions.
-   **Speak Plainly:** Use your own words — Aeriocode will translate them into code.
