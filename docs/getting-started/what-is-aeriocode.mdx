---
title: "What is Aeriocode?"
description: "An introduction to Aeriocode, your AI-powered development assistant in VS Code."
---

Aeriocode is an AI-powered aerospace and engineering development assistant which integrates with Microsoft Visual Studio Code. It provides an interface between your IDE and LLMs facilitating the development of safety-critical, avionics, and engineering software — increasing productivity and lowering the barrier to entry for engineers working on complex systems. Depending on permissions, Aeriocode can read/write files, execute commands, use your web browser, and expand its capabilities with Model Context Protocol servers.

What makes Aeriocode distinctive is its focus on aerospace and engineering workflows combined with a thoughtful approach to code generation. Rather than simply generating code snippets, Aeriocode collaborates with developers by planning solutions step-by-step, maintaining awareness of the entire development environment, and requiring explicit approval for all changes. It understands large codebases written in languages like C, C++, Ada, Python, MATLAB, and Rust — commonly used in avionics, flight dynamics, embedded systems, and simulation environments. Aeriocode accelerates onboarding for new engineers and connects with hundreds of tools through its Model Context Protocol Marketplace, enabling everything from DO-178C traceability workflows to automated hardware-in-the-loop testing—all through natural language commands.



**Aerio Code** is developed based on [<u>**Cline VScode Extension**</u>](https://www.cline.bot).
